System Overview & Architecture

Technology Stack

Layer Technology Purpose
Frontend React 18 + TypeScript + Vite Modern, responsive user interface
Backend Node.js + Express.js + TypeScript RESTful API and business logic
Database PostgreSQL (Neon Serverless) Reliable data persistence
ORM Drizzle ORM Type-safe database interactions
UI Components shadcn/ui + Radix UI Professional component library
Styling Tailwind CSS Utility-first styling framework
State Management TanStack React Query Server state and caching
File Storage Replit Object Storage Secure document & media storage

System Architecture

Three-Tier Architecture

Presentation Layer: React frontend with responsive UI, routing via Wouter, form management with React Hook Form and Zod validation

Business Logic Layer: Express.js API routes with middleware for authentication, authorization, and request validation

Data Layer: PostgreSQL database with Drizzle ORM for type-safe queries, session storage, and audit logging

Deployment & Infrastructure

  • Hosting: Replit (Production) with auto-scaling capabilities
  • Database: Neon serverless PostgreSQL with automatic backups
  • Security: HTTPS/SSL, HTTP-only cookies, CSRF protection, bcrypt password hashing
  • Performance: CDN-delivered static assets, optimized queries, caching strategies

Core Features & Modules

1. Employee Management

Comprehensive employee data management with 360-degree employee profiles, document storage, role assignments, and organizational hierarchy tracking.

Key Features:

  • Employee Profiles: Complete personal, employment, and contact information
  • Multi-Company Support: Manage employees across multiple subsidiaries
  • Document Management: Store contracts, offer letters, ID proofs, tax documents
  • Reporting Manager Assignment: Establish organizational hierarchy and reporting relationships
  • Job Responsibilities: Define and track role-specific duties with document attachments
  • Employment Status Tracking: Active, On Leave, Inactive, Contract, Temporary
  • Bulk Operations: Update multiple employees, bulk document uploads
  • Search & Filtering: Find employees by department, role, status, location

Data Points Tracked:

  • Personal: Name, Email, Phone, DOB, Address, Emergency Contact
  • Employment: ID, Department, Position, Hire Date, Salary Band, Employment Type
  • Hierarchy: Reporting Manager, Direct Reports, Team Leads, Department Leads
  • Documents: Contracts, Offer Letters, ID Proofs, Bank Details, Tax Forms

2. Leave Management System

End-to-end leave request and approval management with multiple leave types, balance tracking, automated approvals, and comprehensive leave history.

Core Functionality:

  • Leave Request Process:
    • Employees request leave via simple form
    • Manager receives notification for approval
    • HR reviews and final approval
    • Automatic balance deduction upon approval
    • Employee receives confirmation
  • Leave Types Supported: Annual, Sick, Casual, Maternity, Paternity, Unpaid, Compensatory, Special
  • Leave Balance Tracking: Real-time balance calculations, carry-over rules, year-end resets
  • Approval Workflow: Multi-stage approvals with comments, rejection with feedback, bulk approvals
  • Balance Adjustments: HR can manually adjust balances for corrections or special circumstances
  • Leave Calendar: Visual representation of approved leaves, team view, department view
  • Compliance: Complete audit trail, leave policy enforcement, regulatory compliance

3. Onboarding System

Structured onboarding process ensuring new hires complete all necessary steps with automated tracking and progress visibility.

Onboarding Stages:

  1. Pre-Onboarding Portal: Public-facing portal for pre-hire information collection
  2. System Access Setup: Credentials, email, system access provisioning
  3. Document Collection: ID, address proof, bank details, declarations
  4. Department Orientation: Introduction to team, processes, tools
  5. Policy Acknowledgment: Read and acknowledge company policies
  6. Psychometric Assessment: Professional personality and aptitude testing
  7. Training Completion: Role-specific training modules
  8. Manager Check-in: Initial performance discussion

HR Admin Capabilities:

  • Create and customize onboarding checklists per role/department
  • Monitor progress across all new hires
  • Send automated reminders for incomplete items
  • Approve/mark items complete on employee behalf
  • Attach documents and instructions
  • Generate onboarding completion reports

4. Project Management
