## ‚úÖ Task Management

### Viewing Your Tasks
1. Click **"Tasks"** in the left sidebar
2. You'll see a list of tasks assigned to you
3. Each task shows:
   - Task title and description
   - Due date and priority level
   - Current status (Pending, In Progress, Completed)
   - Project association (if applicable)

### Working with Tasks
1. **To start a task**: Click on the task and change status to "In Progress"
2. **To complete a task**: Mark it as "Completed" when finished
3. **To add comments**: Use the comment section to communicate with your team
4. **To view details**: Click on any task to see full information

### Task Priorities
- **High**: Red indicator - Complete immediately
- **Medium**: Yellow indicator - Complete within deadline
- **Low**: Green indicator - Complete when time allows
